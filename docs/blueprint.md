# **App Name**: Sepia Dungeon Explorer

## Core Features:

- Player Movement: Enable WASD controls for the player to move the character through the 3D dungeon environment.
- Text Interaction: Detect when the player approaches placeholder objects in corridors and passageways, and display relevant text information in a scroll pop-up on the screen when the enter key is pressed.
- Dungeon Generation: Generate and render a low poly dungeon environment using Three.js, including corridors, secret passageways, and placeholder objects.

## Style Guidelines:

- Apply a sepia tone filter to the entire game to give it an aged look.
- Design a clean and intuitive layout for the scroll pop-up, ensuring readability and ease of interaction.
- Implement smooth camera transitions and character animations to enhance the player's sense of immersion.
- Accent Color: Deep Brown (#4A3026) for interactable elements and highlights.

## Original User Request:
Build a three.js game. the main character is controlled by the WASD buttons. The environment is a dungeon built in low poly style. Exploring different corridors and secret passageways. Add placeholder artwork or objects in these different passage ways that can be approached and on clicking enter will show some text information about it. This text information will show as a scroll pop-up on screen. The entire game should be in sepia tones.
